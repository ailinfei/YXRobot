## 技术架构亮点

### 后端技术栈
- **框架**: Spring Boot 2.7+
- **数据访问**: MyBatis + MySQL
- **验证**: Bean Validation + 自定义验证器
- **异常处理**: 全局异常处理器 + 异常监控
- **文件处理**: 多格式文件上传和处理

### 前端技术栈
- **框架**: Vue 3 + TypeScript
- **状态管理**: Pinia + 组合式API
- **UI组件**: Element Plus
- **路由**: Vue Router 4 + 权限控制
- **工具**: Vite + ESLint + Prettier

### 数据库设计
- **存储引擎**: InnoDB
- **字符集**: UTF-8MB4
- **索引策略**: 复合索引 + 全文索引
- **数据完整性**: 外键约束 + 业务规则
