📋 手动启动步骤

如果一键启动失败，请按以下步骤手动启动：

  1. 打开命令提示符
    按 Win + R，输入 cmd，按回车
  2. 切换到项目目录
    cd /d "E:\YXRobot\workspace\projects\YXRobot"
  3. 启动Spring Boot应用
    mvn -f "E:\YXRobot\workspace\projects\YXRobot\pom.xml" spring-boot:run "-Dspring-boot.run.profiles=dev"
  4. 等待启动完成
    看到 "Started YXRobotApplication" 信息表示启动成功
  5. 验证应用状态
    在浏览器中访问：http://localhost:8081
